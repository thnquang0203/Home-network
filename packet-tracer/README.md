# Thư mục packet-tracer/

Đặt file dự án Packet Tracer thật của bạn vào đây, ví dụ:

```
home-network.pkt
```

Lưu ý: file `.pkt` là định dạng nhị phân do phần mềm Cisco Packet Tracer tạo ra khi bạn bấm **File > Save**. Sau khi dựng xong mạng theo hướng dẫn trong `docs/huong-dan-cau-hinh.md`, hãy lưu file `.pkt` vào đúng thư mục này rồi `git add` trước khi push.

Gợi ý đặt tên file rõ ràng theo phiên bản, ví dụ:
- `home-network-v1.pkt` (bản đầu tiên)
- `home-network-final.pkt` (bản nộp)
